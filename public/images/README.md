# Imágenes Optimizadas para SEO

Todas las imágenes en esta carpeta están optimizadas para SEO con:

- ✅ Nombres descriptivos y SEO-friendly
- ✅ Formato JPG optimizado
- ✅ Alt text descriptivo con palabras clave
- ✅ Títulos informativos
- ✅ Atributos width y height para evitar layout shift
- ✅ Lazy loading para mejor rendimiento

## Estructura de Carpetas

- `castillos-pequenos/` - Castillos hinchables pequeños
- `castillos-grandes/` - Castillos hinchables gigantes  
- `castillos-acuaticos/` - Castillos hinchables acuáticos
- `castillos-tobogan/` - Castillos hinchables con tobogán
- `fiesta-espuma/` - Fiestas de espuma
- `animaciones/` - Animaciones infantiles

## Convención de Nombres

Formato: `categoria-tipo-variante-numero.jpg`

Ejemplos:
- `castillo-hinchable-pequeno-spider-1.jpg`
- `castillo-hinchable-grande-dragonkan-1.jpg`
- `fiesta-espuma-espuma-1.jpg`
